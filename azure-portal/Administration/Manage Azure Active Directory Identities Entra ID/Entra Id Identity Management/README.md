# Azure Project: Manage Azure Active Directory (Microsoft Entra ID) Identities

## Project Level

**Novice | Azure Portal Only**

This project is a beginner-friendly, step-by-step implementation of **Microsoft Entra ID (formerly Azure Active Directory)** identity management using the **Azure Portal**. It is designed for individuals transitioning into **Cloud Support**, **Junior Systems Administrator**, or **Azure Administrator** roles.

The goal is to demonstrate foundational identity and access management skills in a clear, repeatable, and well-documented manner.

---

## Project Objectives

By completing this project, you will demonstrate the ability to:

* Create and manage an Azure AD (Entra ID) tenant
* Create and configure internal Azure AD users
* Create Azure AD security groups with **assigned** and **dynamic** membership
* Invite and manage **guest (B2B)** users

---

## Security & Best Practices Applied

* Use groups instead of direct user permissions
* Follow the principle of least privilege
* Separate administrative and standard users
* Use dynamic groups for scalable access control

---

## Lessons Learned

* Dynamic group membership depends on accurate user attributes
* Guest users have limited permissions by default
* Tenant context switching can cause configuration confusion
